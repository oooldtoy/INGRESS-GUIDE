title: Recharge/充能
description: 使用 XM 为能量塔补充能量
---

在游戏中，能量塔在其被占领之后，每隔 24 小时会损失其满容量 15% 的能量；在受到攻击时能量塔也会损失一定的能量。因此，充电对于维持能量塔的占领状态尤为重要。

# 如何充电

在 40m 范围内，可以通过点击能量塔并选择界面右下角的“充能”按钮进行充电；在 40m 范围外则需要拥有[能量塔钥匙](/INGRESS-GUIDE/reference/key/)，通过点击能量塔钥匙并选择界面左下角的“充能”按钮进行充电。

充电界面有“全部充能”和“全部强力充能”两个按钮。“全部充能”消耗 1000XM 为能量塔进行充电，“全部强力充能”消耗能量槽中所有 XM 为能量塔进行充电；若点击能量塔的某个共振器，则两个按钮将变为“充能”和“强力充能”，此时“全部充能”消耗 1000XM 为该共振器进行充电，“全部强力充能”消耗能量槽中所有 XM 为该共振器进行充电。能量塔增加的能量取决于充能效率与能量塔（或共振器）剩余的能量，若能量塔（或共振器）需要的能量较少，则可能不会消耗 1000XM 或能量槽中所有 XM。需要注意，使用红糖（PowerCube）获得的额外 XM 不属于能量槽中的 XM。

在一次充电过程中，每使目标能量塔获得 100XM，特工就会获得 1AP。如果目标能量塔获得不足 100XM，特工也会获得 1AP。

# 充电范围与效率

充电范围随特工等级的增加而扩大，当特工与能量塔的距离大于充电距离上限时，则无法进行充电操作。具体充电距离上限如下表所示：

<div style="display: flex; justify-content: center; width: 100%;">

| 特工等级  | 能量槽容量 | 充电距离上限  |
| ---- | ---- | ---- |
| L1 | 3000XM | 250km |
| L2 | 4000XM | 500km |
| L3 | 5000XM | 750km |
| L4 | 6000XM | 1000km |
| L5 | 7000XM | 1250km |
| L6 | 8000XM | 1500km |
| L7 | 9000XM | 1750km |
| L8 | 10000XM | 2000km |
| L9 | 11500XM | 2250km |
| L10 | 13000XM | 2500km |
| L11 | 14500XM | 2750km |
| L12 | 16000XM | 3000km |
| L13 | 17500XM | 3250km |
| L14 | 19000XM | 3500km |
| L15 | 20500XM | 3750km |
| L16 | 22000XM | 4000km |

</div>

充能效率随特工与能量塔之间距离的增加而降低。能量塔获得的 XM = 特工消耗的 XM × 充能效率。有趣的是，如果持有能量塔的钥匙（不包括桶中的钥匙），即使充能效率显示为 99.2% 以上，实际充能效率也是 99.2%。
