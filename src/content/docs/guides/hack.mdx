---
title: Hack/入侵
description: 游戏的最基础操作内容
---

Hack是游戏中最基础的操作，它允许你通过触碰并与[Portal](/INGRESS-GUIDE/reference/portal/)交互来为你生成游戏中的道具。

你可以利用下述的五种方式对[Portal](/INGRESS-GUIDE/reference/portal/)进行Hack。


import { Card, CardGrid } from '@astrojs/starlight/components';

<CardGrid>
  <Card title="便捷Hack（无需key）" icon="left-arrow">
    ![](../../../assets/hack5.png)通过长按Portal向右上滑动以实现无需key的快捷hack。
  </Card>
  <Card title="便捷Hack（需要key）" icon="right-arrow">
    ![](../../../assets/hack4.png)通过长按Portal向左上滑动以实现需要key的快捷hack。
  </Card>
  <Card title="Hack（无需key）" icon="close">
    ![](../../../assets/hack3.png)点击进入Portal详情页后，长按Hack按钮后滑向右上角以完成需求key的Hack。
  </Card>
  <Card title="Hack（需要key）" icon="approve-check">
    ![](../../../assets/hack2.png)点击进入Portal详情页后，长按Hack按钮后滑向上方以完成无需key的Hack*。
  </Card>
</CardGrid>
\* 目前有报道称Hack（需要key）的操作方法会导致无法正常获取key，需要key的时候请避免使用这种方式。
  <CardGrid>
  <Card title="Glyph Hack" icon="approve-check">
    ![](../../../assets/hack1.png)点击进入Portal详情页后，长按Hack按钮后滑向左上方进入[Glyph Hack](/INGRESS-GUIDE/gameplay/glyph-hack/)以获取更多的物资。
  </Card>
  </CardGrid>
