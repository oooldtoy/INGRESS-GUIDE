### 什么是 Glyph Hack？

Glyph Hack（画图Hack）是在 Portal 界面上按住“HACK”按钮左滑触发的一个记忆小游戏。通过在倒计时内按顺序正确画出所有出现的图形符号（Glyph）且每个图形都需要一笔画完，完成后玩家可以获得更多物资和额外 AP 奖励。

Hack等级（即玩家等级与Portal等级中的较小值）不同时，画图小游戏一开始会显示的图形数量和最后完成游戏的时限不同，具体见下表：

<div style="display: flex; justify-content: center; width: 100%;">

| Hack等级 | Glyph数量 | 完成时限  |
| ---- | ---- | ---- |
| L1 | 1 | 20s |
| L2 | 2 | 20s |
| L3 | 3 | 20s |
| L4 | 3 | 19s |
| L5 | 3 | 18s |
| L6 | 4 | 17s |
| L7 | 4 | 16s |
| L8 | 5 | 15s |

</div>

你可以使用下方的画板画出熟悉 Glyph 符号。如果这是一个存在的Glyph，它会显示出该Glyph的名字，并列出所有以它开头的画图序列（Sequence）。
