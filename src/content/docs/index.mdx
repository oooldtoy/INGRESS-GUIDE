## 世界观与阵营

瑞士日内瓦的 CERN（欧洲核子研究中心）在大型强子对撞机实验中意外发现了一种全新的物质——**Exotic Matter（XM）**。这种与人类思想、创造力直接相关的超维能量，正通过世界各地的传送门涌入地球，其背后似乎隐藏着名为「Shapers（塑形者）」的高维地外智能。

面对 XM 力量带来的巨变，人类分化为两大阵营，你必须选择其中之一：

- 🟢 **启蒙军（Enlightened，绿军）**：相信 XM 是推动人类心智进化的催化剂，试图接纳和运用这些能量，引导人类走向更高的境界。
- 🔵 **反抗军（Resistance，蓝军）**：视未经证实的 XM 能量为对人类自由意志的潜在威胁，主张人类应该保持自主，抵御未知力量的入侵。

此外，你还会遇到脱离阵营控制、自行增殖的红色未知力量 🔴 **[Machina](/INGRESS-GUIDE/reference/machina/)**。

## 探员快速上手指引

作为新晋特工，你可以遵循以下基本行动路线：

1. **基础生存：** 了解你的 [Scanner（游戏界面）](/INGRESS-GUIDE/guides/scanner/)，在地图上找到附近发光的 [Portal（能量塔）](/INGRESS-GUIDE/reference/portal/)，尝试进行 [Hack（入侵）](/INGRESS-GUIDE/guides/hack/) 以获取物资。
2. **阵地建设：** 向空白或友方的传送门 [部署共振器](/INGRESS-GUIDE/guides/deploy/)，然后为它安装 [Mods（强化模组）](/INGRESS-GUIDE/reference/mods/)。
3. **连线建场：** 运用你入侵获得的 [Key（钥匙）](/INGRESS-GUIDE/reference/key/)，将两个满共振器的传送门进行 [Link（连线）](/INGRESS-GUIDE/guides/link/)。将三个传送门互相连结形成三角形，你就能建立起一块 [Control Field（控制场）](/INGRESS-GUIDE/guides/controlfield/)。
4. **摧毁与占领：** 遇到敌方阵营的传送门？使用 [XMP](/INGRESS-GUIDE/reference/xmp/) 摧毁它们，把它抢回来！

## 行动准则

- **安全第一**：始终注意现实世界中的车辆与环境，遵守当地法规。
- **长线作战**：在 Ingress 中，[经验值](/INGRESS-GUIDE/data/accesspoint/) 的积累是一个漫长的过程。合理运用 [多重技巧](/INGRESS-GUIDE/gameplay/multifield/) 可以加快你的升级速度。
- **玩家合作**：Ingress 的高阶玩法高度依赖玩家合作，例如建造覆盖整座城市的 [BAF（大面积控制场）](/INGRESS-GUIDE/gameplay/baf/) 或参与线下的全球 [XM 异变活动（Anomaly）](/INGRESS-GUIDE/gameplay/xmanomaly/)。
