模组是一系列用于增强Portal性能的可部署模块。Mod的部署可参考[部署](/INGRESS-GUIDE/guides/deploy/)。

模组包括以下几个类别：

<Tabs>
  <TabItem label="防御性模组">防御性模组主要为Portal提供[防御值](/INGRESS-GUIDE/reference/migitation/)，包括三种不同稀有度的Shield以及一种特殊的高级Shield，参见[Shield](/INGRESS-GUIDE/reference/shield/)。</TabItem>
  <TabItem label="进攻性模组">进攻性模组主要包括Force Amp和Turret，分别可以增强Portal反击敌人攻击的攻击力及攻击频率。<br />~~通常被当作垃圾Mod随意乱插。~~</TabItem>
  <TabItem label="连线增强器">主要用于增幅Portal的可连接距离，详情参见[连接增强器](/INGRESS-GUIDE/reference/linkamp/)。<br />属于连线增强器的[Softbank Ultralink](/INGRESS-GUIDE/reference/linkamp/#softbank-ultralink)有特殊功能，例如增加[外连Link上限](/INGRESS-GUIDE/data/outlinkcap/)及增加Link附加的[防御值](/INGRESS-GUIDE/reference/migitation/)。</TabItem>
  <TabItem label="生产模组">生产模组主要用来改变Portal的Hack冷却时间、冷却次数或改变产出物资的种类，详情参见[生产模组](/INGRESS-GUIDE/reference/farmmod/)。</TabItem>
</Tabs>
