Beacon（信标）是一种可部署在 [Portal](/INGRESS-GUIDE/reference/portal/) 上的消耗品。部署后，Portal 上方会出现一个醒目的图案动效，在 Scanner 地图上远距离即可看见，方便用于标记特定 Portal。

### Beacon 的获取

Beacon 无法通过普通 [Hack](/INGRESS-GUIDE/guides/hack/) 获得，主要来源包括：

- **游戏商店（Store）**：通过充值购买，单个或套装形式出售。
- **活动奖励**：部分官方活动、纪念日活动或比赛会以 Beacon 作为奖品发放。
- **Passcode 兑换**：特定活动的 Passcode 有时包含 Beacon。

<Aside type="tip">
Beacon 属于付费/稀有道具，使用前请确认目的，避免浪费。
</Aside>

### Beacon 的种类

Beacon 有多种外观图案和功能，主要类型包括：

| 类型 | 说明 | 持续时间 |
| :--- | :--- | :--- |
| **基础 Beacon** | 商店通用版本，有多种配色图案可选<br /><Image src={beaconImg} alt="Beacon" width={60} /> | 4小时 |
| **阵营 Fireworks** | 阵营专属信标（启蒙军和反抗军），部署后呈现阵营标志图案<br /><Image src={fireworkEnlImg} alt="ENL Fireworks" width={60} /> <Image src={fireworkResImg} alt="RES Fireworks" width={60} /> | 10分钟 |
| **Battle Beacon** | 战斗信标分为 Rare (稀有) 和 Very Rare (非常稀有) 两种。用于触发 Portal 对抗事件，详情请参考 [Battle Beacon](/INGRESS-GUIDE/reference/battlebeacon/) 页面。<br /><Image src={rareBbImg} alt="Rare Battle Beacon" width={60} /> <Image src={veryRareBbImg} alt="VR Battle Beacon" width={60} /> | 视阶段而定 |

### Beacon 的部署

要部署 Beacon，你需要靠近目标 Portal 直至进入互动范围（40米圈内），然后：

1. 打开主界面的 **Inventory（物品栏）**
2. 找到你想使用的 Beacon 道具
3. 点击 **USE**
4. 在地表视图上点击确认你想部署的目标 Portal

部署后 Beacon 即刻生效，**立即消耗**无法取回。成功部署 Beacon 会给予玩家 **500 AP** 奖励。

### 持续时间

不同类型的 Beacon 持续时间不同。普通基础 Beacon 的存在时间为 **4小时**，而阵营 Fireworks 的存在时间只有 **10分钟**。时间耗尽后图案自动消失，Portal 恢复正常外观。

每个 Portal 同一时间只能存在一个 Beacon（Battle Beacon 除外）；若重复部署，新的 Beacon 会覆盖旧的。

<Aside type="caution">
基础 Beacon 和 Fireworks 不影响任何游戏数值，不提供 AP，也不改变 Portal 的 Hack 产出或防御能力，仅为视觉标记效果。
</Aside>

### Portal Fracker

[Portal Fracker（破解器）](/INGRESS-GUIDE/reference/fracker/) 虽然在背包中与 Beacon 放在一起，且部署方式相似，但它是**完全不同的功能性道具**。

- 部署 Fracker 会大幅提高 Portal 的物资掉落（双倍产出），是用于建立“闪建农场”的核心道具。
- Fracker 有明确的副作用和数量/时间限制。

详细机制与使用方法，请查看 [Portal Fracker 专用页面](/INGRESS-GUIDE/reference/fracker/)。
