Beacon（信标）是一种可部署在 [Portal](/INGRESS-GUIDE/reference/portal/) 上的消耗品。部署后，Portal 上方会出现一个醒目的图案动效，在 Scanner 地图上远距离即可看见，方便用于标记特定 Portal。

### Beacon 的获取

Beacon 无法通过普通 [Hack](/INGRESS-GUIDE/guides/hack/) 获得，主要来源包括：

- 可以通过充值在游戏商店里购买，单个或套装形式出售。
- 部分官方活动、纪念日活动或比赛会以 Beacon 作为奖品发放。
- 通过[动能桶](/INGRESS-GUIDE/reference/kineticcapsule/)合成。

### Beacon 的部署

要部署 Beacon，你需要靠近目标 Portal 直至进入互动范围（40米圈内），然后：

1. 打开主界面的 **Inventory（物品栏）**
2. 找到你想使用的 Beacon 道具
3. 点击 **USE**
4. 在地表视图上点击确认你想部署的目标 Portal

部署后 Beacon 即刻生效，**立即消耗**无法取回。成功部署 Beacon 会给予玩家 **500 AP** 奖励。

### 持续时间

不同类型的 Beacon 持续时间不同。普通基础 Beacon 的存在时间为 **4小时**，而阵营 Fireworks 的存在时间只有 **10分钟**。时间耗尽后图案自动消失，Portal 恢复正常外观。

每个 Portal 同一时间只能存在一个 Beacon，因此我们通常称Beacon也存在“免疫”，即在目前已经存在一个Beacon的前提下无法部署另一个Beacon。

烟花（Fireworks）不受该限制：即使Beacon正在生效，也可以再插上一根烟花。

<Aside>
基础 Beacon 和 Fireworks 不影响任何游戏数值，不提供 AP，也不改变 Portal 的 Hack 产出或防御能力，仅为视觉标记效果。
</Aside>

### Portal Fracker

[Portal Fracker（薯条）](/INGRESS-GUIDE/reference/fracker/) 虽然在背包中与 Beacon 放在一起，且部署方式相似，但它是**完全不同的功能性道具**。

- 部署 Fracker 会大幅提高 Portal 的物资掉落（双倍产出），通常在[起8](/INGRESS-GUIDE/reference/portal/#portal%E7%9A%84%E7%AD%89%E7%BA%A7)或需要快速大量Hack钥匙时使用。
- Fracker 有明确的副作用和数量/时间限制。

详细机制与使用方法，请查看 [Portal Fracker](/INGRESS-GUIDE/reference/fracker/)。
