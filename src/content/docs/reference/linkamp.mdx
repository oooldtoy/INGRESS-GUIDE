Link Amp是游戏中一类能够增加出射Link距离的Mod，截止目前，一共有三个稀有度不同的道具。
同一个Portal上部署多个Link Amp可以进一步增加可出射Link距离上限，但倍率随叠加个数[显著衰减](/INGRESS-GUIDE/data/diminishingrule/)。

### Rare Link Amp

<div style="display: flex; justify-content: center; width: 100%;">
<Image src={rla} width={200} alt="" />
</div>

稀有等级的Link Amp提供的增幅倍率为2倍，并不提供其他增益。

### SoftBank Ultralink

<div style="display: flex; justify-content: center; width: 100%;">
<Image src={sbul} width={200} alt="" />
</div>

Softbank Ultralink（SBUL）虽然不叫Link Amp，但是同样提供连接距离上限的加成，增幅倍率为5倍。
区别于其他的Link Amp，Softbank Ultralink还提供以下增益：
- 提高[出射连接数量上限](/INGRESS-GUIDE/data/diminishingrule/#%E5%87%BA%E5%B0%84%E4%B8%8A%E9%99%90)8条。
- 将Link提供的[防御值](/INGRESS-GUIDE/reference/mitigation/)提高至1.5倍。该属性同样会随同Portal上SBUL的数量增加而衰减。
<Aside>
虽然SBUL可以提供一些防御力，但是并不建议通过SBUL来防御被激烈进攻的Portal，即使其上有很多连线。主要原因在于SBUL的[黏性](/INGRESS-GUIDE/reference/stickiness/)低于高级[Shield](/INGRESS-GUIDE/reference/shield/)，更容易被破坏。
</Aside>

### Very Rare Link Amp
<div style="display: flex; justify-content: center; width: 100%;">
<Image src={vrla} width={200} alt="" />
</div>

非常稀有等级的Link Amp是游戏中最为稀缺的道具，不存在于[Hack](/INGRESS-GUIDE/guides/hack/)产出中，只在一些特定的活动中以[Passcode](/INGRESS-GUIDE/guides/passcode/)的形式分发。

其提供的连线长度增幅倍率为7倍，为Link Amp中最高的倍率。

虽然其稀有性毋庸置疑，但是日常活动中可谓是完全没用，只在做一些[非常大的大Field](/INGRESS-GUIDE/gameplay/baf/)的时候才有可能会用到。
