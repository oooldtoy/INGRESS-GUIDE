Link Amplifier是游戏中一类能够增加出射Link距离的Mod，截止目前，一共有三个稀有度不同的道具。
同一个Portal上部署多个Link Amplifier可以进一步增加可出射Link距离上限，但倍率随叠加个数[显著衰减](/INGRESS-GUIDE/data/diminishingrule/)。
